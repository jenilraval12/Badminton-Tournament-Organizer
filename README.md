## 🚀 Features
- User Signup / Signin
- Start Tournament
- Continue Tournament
- Quick Match Mode
- Quick Game Mode
- Match Statistics
- MySQL Database Integration

## 🛠 Tech Stack
- Python
- MySQL
- VS Code
